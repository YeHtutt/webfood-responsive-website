# 🍔 Webfood Responsive Website

A responsive food ordering website built as part of the **100 Days of Web Development** course.

In addition to the course requirements, I implemented a custom **animated hamburger menu** using JavaScript to improve the mobile navigation experience.

## 🚀 Live Demo

https://yehtutt.github.io/webfood-responsive-website/

## 📂 Repository

https://github.com/YeHtutt/webfood-responsive-website

## 📖 About the Project

This project focuses on building a responsive website that adapts to different screen sizes and devices.

The website includes:

* Responsive page layout
* Mobile-friendly navigation
* CSS Flexbox layouts
* Media queries
* Interactive JavaScript functionality
* Custom animated hamburger menu

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 Learning Objectives

* Build responsive websites
* Use media queries effectively
* Create mobile-first layouts
* Improve navigation for smaller screens
* Add interactive UI elements with JavaScript
